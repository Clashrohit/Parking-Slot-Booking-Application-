# Parking Slot Booking Application

A Flutter application for booking parking slots. This app allows users to view and reserve parking spaces in a grid layout.

## Features

- Display 10 parking slots in a grid view
- Toggle slot availability (green for available, red for occupied)
- Simple and intuitive user interface

## Getting Started

### Prerequisites

- Flutter SDK installed on your machine
- Android Studio or VS Code with Flutter extensions
- A device or emulator for testing

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/Clashrohit/Parking-Slot-Booking-Application-.git
   ```

2. Navigate to the project directory:
   ```
   cd newparkingslot
   ```

3. Install dependencies:
   ```
   flutter pub get
   ```

4. Run the app:
   ```
   flutter run
   ```

## Project Structure

- `lib/main.dart`: Entry point of the application
- `lib/parking_slot.dart`: Main widget for the parking slot grid
- `android/`: Android-specific configurations
- `ios/`: iOS-specific configurations
